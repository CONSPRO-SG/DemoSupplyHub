# Consumable Control

Industrial Consumable Control application built with:

- Next.js (front-end and API routes)
- Convex backend (database + functions)
- Convex Auth (password sign-in / sign-up)

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Copy environment variables:
   ```bash
   cp .env.example .env
   ```
3. Start Convex backend:
   ```bash
   npm run convex:dev
   ```
4. Start app:
   ```bash
   npm run dev
   ```
5. Open the app and create an account (Sign Up), then sign in.
